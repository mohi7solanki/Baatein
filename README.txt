Baatein - Chat App
------------------

Stack used - MEVN stack

  Steps to run the project
  ------------------------
  - Install `npm` and `mongod` service globally in your system.
  - Install packages locally in both client as well as server folder by running `npm install`.
  - Start mongo service by running `sudo mongod`.
  - Go to client folder and run `npm run serve`.(Ignore errors and warnings - some lint issues.)
  - Go to server folder and run  `npm start`.
  - Now the localhost is up and running on 8080(if not busy else on other port), open the url in your browser.
  - Enjoy your baatein.
